3. Exponential Regression Model

$$
f(x) = e^{w_0x}
$$

$$
g(x) = \ln f(x) = w_0x
$$

  • Exponential 의 경우, Linear Model 형태를 만들어 주기 위해, log 를 씌워 주워 train 을 시킨후, 다시 exponential 을 양변에 취해주는 형태로 modeling 을 하여야 한다.
